## How to use

Clone the repo to your folder:

```bash
mkdir nextjs-pwa
cd nextjs-pwa
git clone https://github.com/gufranmirza/nextjs-apollo-grapghql-pwa.git
```

Install it and run:

```bash
yarn
yarn run dev
```

And that's all!
